# FlickLy
FlickLy is a practical simple implementation of MVVM in a SwiftUI

# How to run?
- Goto [The Movie DB site](https://themoviedb.org)
- Create your API key
- Expand `AppEnvironment` group
- Select `AppConstants.swift`
- Fill in `struct TheMovieDBAppId`
